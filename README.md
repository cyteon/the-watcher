# The Watcher

An uptime monitor made in solidstart and sqlite w/ drizzle.

## Setup

1. Clone the repository:
```bash
> git clone https://github.com/cyteon/the-watcher
```

2. Install deps:
```bash
> npm i
```

3. Create db:
```bash
> npx drizzle-kit push
```

4. Run in dev:
```bash
> npm run dev
```
